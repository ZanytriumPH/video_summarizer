# 使用指南

## 快速开始

### 1. 首次运行

```bash
# 进入项目目录
cd flutter_project

# 获取依赖
flutter pub get

# 运行应用
flutter run
```

## 应用功能导览

### 底部导航栏

应用底部有5个选项卡，可通过点击切换：

1. **⚙️ SETTINGS** - 设置和配置
2. **🎬 VIDEO** - 视频选择
3. **⏳ PROGRESS** - 处理进度
4. **📝 SUMMARY** - 总结结果
5. **🔎 Q&A** - 时间旅行问答

### 屏幕功能详解

#### ⚙️ Settings 屏幕

**功能**：
- 配置OpenAI API Key
- 设置Base URL
- 输入总结偏好
- 选择并行处理模式

**操作步骤**：
1. 在 "OpenAI API Key" 字段输入你的API密钥
2. 如需修改，在 "Base URL" 字段编辑（默认为OpenAI官方）
3. 在 "总结偏好" 中输入你想要总结的重点方向
4. 在 "并行模式" 中选择处理模式
5. 点击 "保存配置" 按钮

**提示**：
- API Key会保存到设备本地存储
- 密码输入框会隐藏你的密钥内容

#### 🎬 VIDEO Selection 屏幕

**功能**：
- 输入YouTube视频链接
- 选择本地视频文件

**操作步骤**：
1. **使用YouTube**：
   - 选择 "🌐 YouTube URL" 选项
   - 输入视频链接
   - 点击 "下一步" 开始处理

2. **上传本地视频**：
   - 选择 "📤 本地上传" 选项
   - 点击上传区域选择视频文件
   - 点击 "下一步" 开始处理

**提示**：
- 同时只能选择一个视频源
- 下一步按钮只在选定有效源时启用

#### ⏳ PROGRESS 屏幕

**功能**：
- 实时显示视频处理进度
- 查看各模块分析进度

**显示内容**：
- **整体进度**：总处理百分比
- **音频分析**：音频内容提取进度
- **视觉分析**：图像识别进度
- **融合综合**：多模态信息融合进度

**提示**：
- 进度条实时更新
- 处理完成后自动跳转到结果屏幕

#### 📝 SUMMARY Result 屏幕

**功能**：
- 查看完整的视频分析总结
- 查看核心要点
- 查看音频和视觉亮点

**显示内容**：
- 视频标题
- 核心要点（通常3-5个）
- 音频摘要
- 视觉亮点
- 完整分析内容

**操作**：
- 点击 "复制全文" 将总结复制到剪贴板
- 点击 "分享报告" 分享总结（功能开发中）

**提示**：
- 可以向下滚动查看完整内容
- 生成时间显示在底部

#### 🔎 TIME TRAVEL Q&A 屏幕

**功能**：
- 按时间戳追问视频内容
- 查看问答历史

**操作步骤**：
1. 在 "时间戳" 字段输入目标时间（格式：MM:SS，如 1:30）
2. 使用 "证据窗口" 滑块选择时间范围（5-60秒）
3. 在 "追问问题" 字段输入你的问题
4. 点击 "追问问题" 按钮提交

**示例**：
- 时间戳：2:30
- 证据窗口：20秒
- 问题：这段时间讲的是什么？

**提示**：
- 时间戳应该在视频长度范围内
- 证据窗口会围绕指定时间戳扩展
- 问答历史显示在底部，可向上滚动查看

## 常见操作

### 保存配置

在Settings屏幕输入配置后，点击 "保存配置" 按钮。系统会：
- 保存API Key到本地设备
- 保存Base URL
- 保存总结偏好
- 保存并行模式选择

### 处理视频

1. 进入VIDEO屏幕
2. 选择视频源（YouTube或本地）
3. 点击"下一步"
4. 应用会自动跳转到PROGRESS屏幕
5. 查看实时处理进度
6. 处理完成后查看SUMMARY结果

### 查询特定内容

1. 进入Q&A屏幕
2. 输入要查询的时间戳
3. 调整证据窗口大小
4. 输入具体问题
5. 查看回答结果

## UI设计特点

### 颜色系统

- **蓝色（#0066FF）**：主要操作、高亮、活跃状态
- **灰色（#F5F5F5）**：背景色
- **深灰（#1A1A1A）**：主文本
- **浅灰（#999999）**：辅助文本
- **黄色（#FFF8E1）**：提示/结果背景

### 交互设计

- **卡片式布局**：清晰的内容分离
- **圆角边框**：现代化外观
- **实时反馈**：按钮启用/禁用状态
- **滑块选择**：范围选择的直观方式

## 键盘快捷键（开发模式）

当在模拟器/真机上运行 `flutter run` 时：

- **r** - 热重载（保持应用状态）
- **R** - 完全重启应用
- **q** - 退出应用
- **p** - 切换性能分析
- **w** - 打印widget树

## 故障排除

### 问题：配置无法保存

**解决方案**：
1. 检查是否点击了 "保存配置" 按钮
2. 查看屏幕底部是否显示 "配置已保存" 提示

### 问题：视频处理卡住

**解决方案**：
1. 检查网络连接
2. 检查API Key是否正确
3. 检查Base URL是否正确

### 问题：时间戳无法输入

**解决方案**：
1. 确保输入格式为 MM:SS
2. 确保时间在视频长度范围内

### 问题：看不到历史问答

**解决方案**：
1. 向上滚动屏幕查看
2. 确保已提交过问题

## 性能提示

- 首次加载时会初始化应用状态
- 长视频处理可能需要较长时间
- 在WiFi环境下运行可获得最佳性能
- 关闭其他应用可提高处理速度

## 隐私和安全

- API Key保存在本地设备
- 所有视频数据通过加密连接传输
- 可以随时在Settings屏幕修改或删除API Key
