# 视频总结器-移动端设计说明

## 设计概述
这是一个多屏幕移动应用设计，采用React Native技术栈，为iOS/Android平台优化。

## 屏幕列表

### 1️⃣ Settings（配置屏幕）
**功能**：配置API Key、选择并行模式等基本设置
- **位置**：活跃标签栏第一个图标 ⚙️
- **主要元素**：
  - API Key输入框（密码输入）
  - Base URL输入框
  - 总结偏好文本区
  - 并行模式单选（ThreadPool/Send API）

### 2️⃣ Video Selection（视频选择屏幕）
**功能**：选择视频来源（YouTube或本地上传）
- **位置**：活跃标签栏第二个图标 🎬
- **主要元素**：
  - YouTube URL输入选项（蓝色高亮，表示默认选中）
  - 本地上传选项
  - 拖拽上传区域
  - 下一步按钮

### 3️⃣ Processing Progress（处理进度屏幕）
**功能**：实时显示视频处理的多模态分析进度
- **位置**：活跃标签栏第三个图标 ⏳
- **主要元素**：
  - 总体进度条与百分比
  - 音频分析进度卡片 (🎵)
  - 视觉分析进度卡片 (👁️)
  - 融合综合进度卡片 (🔗)
  - 每个模块都有独立的进度条

### 4️⃣ Summary Result（总结结果屏幕）
**功能**：展示完整的视频分析总结报告
- **位置**：活跃标签栏第四个图标 📝
- **主要元素**：
  - 可滚动的总结内容区（核心要点、音频摘要、视觉亮点）
  - 复制全文按钮
  - 分享报告按钮

### 5️⃣ Time Travel Q&A（时间旅行追问屏幕）
**功能**：按指定时间戳追问视频内容
- **位置**：活跃标签栏第五个图标 🔎
- **主要元素**：
  - 时间戳输入 (MM:SS格式)
  - 证据窗口滑块 (5-60秒)
  - 追问问题输入区
  - 追问问题按钮
  - 结果预览卡片（黄色背景）

## 视觉风格

### 色彩系统
- **主色调**：#0066FF（蓝色，CTA和活跃状态）
- **背景**：#F5F5F5（浅灰色）
- **文本主色**：#1A1A1A（深灰）
- **文本辅色**：#666666（中灰）
- **文本弱色**：#999999（浅灰）
- **分隔线**：#DDDDDD（浅灰边框）
- **结果高亮**：#FFF8E1（浅黄背景）

### 字体系统
- **标题**：24px, fontWeight: 700
- **副标题**：12px, fontWeight: 400
- **标签**：13px, fontWeight: 600
- **正文**：12px, fontWeight: 400
- **底部导航标签**：10px, fontWeight: 600, 大写

### 间距系统
- **屏幕顶部过滤条纹**：62px (状态栏)
- **屏幕底部过滤条纹**：62px (导航栏)
- **内容区间隙**：16-20px (水平内边距)
- **内容块间距**：16-20px (垂直间距)
- **卡片内间距**：12-16px

### 圆角
- **大圆角**：12px (卡片、主容器)
- **中圆角**：8px (输入框、副容器)
- **快速导航圆角**：36px (导航栏外壳)
- **导航项圆角**：26px (导航项)

## 底部导航栏设计
- **布局**：5个标签项水平排列在圆形胶囊形容器内
- **活跃状态**：蓝色填充背景 (#0066FF) + 白色文本
- **非活跃状态**：透明背景 + 灰色文本 (#999999)
- **图标 + 标签**：每个标签由emoji图标（18px）+ 大写文本标签（10px）组成
- **间距**：标签间4px间距

## 交互特性
- **屏幕切换**：点击底部导航栏的对应标签切换屏幕
- **滚动行为**：每个屏幕内容超过视口时可垂直滚动
- **进度条动画**：Processing屏幕中的进度条实时更新
- **输入框反馈**：获得焦点时显示焦点状态

## 响应式设计
- **基础宽度**：428px (iPhone 12/13标准宽度)
- **基础高度**：926px (iPhone 12/13标准高度)
- **安全区考虑**：状态栏顶部(62px)、Home指示器底部(21px)

## 设计文件信息
- **格式**：.pen (Pencil设计文件)
- **屏幕数量**：5个完整屏幕
- **排列方式**：水平布局，从左到右排列，便于同步审查
